# Google Ads Campaign Analysis

**Data Analytics Course Campaign | Hyderabad | 2024**

An end-to-end marketing analytics project that transforms raw Google Ads campaign data into actionable business insights using Python and Pandas.

The project covers the complete analytics workflow, from data cleaning and feature engineering to performance analysis and business recommendations.

---

## Project Structure

### Notebook 1 — Data Cleaning

The raw dataset contained multiple quality issues that needed to be resolved before analysis.

Tasks performed:

* Handled missing values
* Removed currency symbols from numeric columns
* Fixed mixed date formats
* Corrected incorrect data types
* Standardized categorical values
* Performed column-by-column validation and cleaning

---

### Notebook 2 — Campaign Analysis

Performed exploratory and business-focused analysis to identify performance drivers and optimization opportunities.

Analysis included:

* Monthly performance trends
* Device segmentation
* Keyword performance analysis
* Day-of-week performance analysis
* ROAS and profitability analysis
* November revenue spike investigation
* Business recommendations

---

## Key Findings

* ROAS = **6.97x**, indicating a highly profitable campaign
* November contributed **76% of annual revenue**, creating concentration risk
* Mobile was the most cost-efficient device
* Thursday generated the highest conversions
* Saturday delivered the strongest ROAS
* Tuesday was the weakest-performing day
* Higher click volume did not necessarily result in higher conversion rates

---

## Business Recommendations

* Increase budget allocation toward Mobile campaigns
* Increase bids for high-performing keywords
* Allocate more budget to Thursday and Saturday campaigns
* Reduce spend on low-performing days such as Tuesday
* Reduce dependency on November by improving performance during slower months
